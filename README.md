# tennis_atp_data

## Description
This dataset is designed to be a continuation and more lightweight implementation of [Jeff Sackmann's](https://github.com/JeffSackmann/tennis_atp) atp dataset. Only including relevant matches 2021 - present(Excluding qualifiers & going forwards will be excluding Davis Cup matches).

## Update Schedule
Expect updates for the 2026 season to come weekly after tournament end. Individual CSV files will also be uploaded for each individual tournament alongside a master dataset.
